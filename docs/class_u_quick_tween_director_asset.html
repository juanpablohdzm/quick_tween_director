Detailed Description

Serializable asset that stores a complete tween-sequence definition.

Create one from the Content Browser (right-click → QuickTween → Director Sequence). Open it by double-clicking to launch the visual timeline editor. At runtime, pass the asset to UQuickTweenDirectorPlayer to play the animation.
