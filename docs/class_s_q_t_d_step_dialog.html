Detailed Description

Modal-style dialog for editing a single FQTDStepData. Opened by double-clicking a step box in the timeline.

Uses UE's property editor (IDetailsView) to show the step's UPROPERTY fields via a thin UObject wrapper, then writes changes back to the asset on confirm.
