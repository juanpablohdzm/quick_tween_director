\doxysubsection{Detailed Description}
Modal-\/style dialog for editing a single \doxylink{struct_f_q_t_d_step_data}{FQTDStep\+Data}. Opened by double-\/clicking a step box in the timeline.

Uses UE\textquotesingle{}s property editor (IDetails\+View) to show the step\textquotesingle{}s UPROPERTY fields via a thin UObject wrapper, then writes changes back to the asset on confirm.
