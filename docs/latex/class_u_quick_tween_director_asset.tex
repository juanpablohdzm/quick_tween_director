\doxysubsection{Detailed Description}
Serializable asset that stores a complete tween-\/sequence definition.

Create one from the Content Browser (right-\/click → Quick\+Tween → Director Sequence). Open it by double-\/clicking to launch the visual timeline editor. At runtime, pass the asset to \doxylink{class_u_quick_tween_director_player}{UQuick\+Tween\+Director\+Player} to play the animation.
