\doxysubsection{Detailed Description}
Runtime player that instantiates and plays a \doxylink{class_u_quick_tween_director_asset}{UQuick\+Tween\+Director\+Asset}.

Workflow (Blueprint)\+:
\begin{DoxyEnumerate}
\item Call UQuick\+Tween\+Director\+Library\+::\+Create\+Director\+Player(\+Asset, self)
\item Call Bind\+Slot(Name, Actor/\+Component) for every named slot in the asset.
\item Call Play() — the player builds the tweens on first play.
\item Optionally add the player to another sequence with UQuick\+Tween\+Library functions, just like any UQuick\+Tweenable.
\end{DoxyEnumerate}

Implements the full UQuick\+Tweenable interface so it behaves identically to a UQuick\+Tween\+Sequence and can be nested inside other sequences.
