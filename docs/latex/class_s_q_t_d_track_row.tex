\doxysubsection{Detailed Description}
One track row in the director timeline.

This widget shows ONLY the label column (component name, icon, delete button). The step content widget is hosted separately (in the shared horizontal scroll box) and can be retrieved via \doxylink{class_s_q_t_d_track_row_a84e836c3e60a264061dea128375af624}{Get\+Step\+Content\+Widget()}.

Step interactions (on the step content widget)\+:
\begin{DoxyItemize}
\item Left-\/click + drag → move (change Start\+Time)
\item Double-\/click → open the step edit dialog
\item Right-\/click empty → context submenu with animation types filtered by component class
\item Right-\/click step → delete option 
\end{DoxyItemize}
