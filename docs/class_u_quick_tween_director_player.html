Detailed Description

Runtime player that instantiates and plays a UQuickTweenDirectorAsset.

Workflow (Blueprint):

  1. Call UQuickTweenDirectorLibrary::CreateDirectorPlayer(Asset, self)
  2. Call BindSlot(Name, Actor/Component) for every named slot in the asset.
  3. Call Play() — the player builds the tweens on first play.
  4. Optionally add the player to another sequence with UQuickTweenLibrary functions, just like any UQuickTweenable.

Implements the full UQuickTweenable interface so it behaves identically to a UQuickTweenSequence and can be nested inside other sequences.
